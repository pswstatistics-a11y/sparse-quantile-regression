# High-Dimensional Sparse Quantile Regression via Proximal Gradient Descent

본 저장소는 오차항의 정규성 가정이 배제된 고차원 다중공선성 환경($P \ge N$) 하에서 전통적인 $L_2$ 규제 추정량(Lasso)의 수리적 취약성을 극복하기 위해, $L_1$-벌점화 분위수 회귀 모형을 구축하고 몬테카를로 시뮬레이션을 통해 수치적 거동을 분석한 연구 레포지토리입니다.

---

## 핵심 기술 보고서 (Technical Report)
본 보고서의 알고리즘, 가우시안/코시/지수 오차항 하에서의 100회 복제 스트레스 테스트 결과, 그리고 차원 확장에 따른 희소성 제어 고찰은 루트 경로의 기술 사양서에 수록되어 있습니다.
[최종 PDF 보고서 읽기 (report.pdf)](./report.pdf) (LaTeX 원문은 `report.tex`를 참조하십시오.)

## 주요 방법론 명세 (Methodology)

### 1. 목적함수 및 수치 최적화 (Numerical Optimization)
체크 손실 함수와 $L_1$ 벌점항의 결합으로 인해 발생하는 목적함수의 비미분성을 처리하기 위해, 손실 함수와 벌점항을 분리하여 연산하는 근접 경사하강법 알고리즘을 직접 유도하여 적용했습니다. 미세한 오차 추정치들을 0으로 수축시키기 위해 수치 해석적 소프트 임계값 연산자를 독립 솔버 레이어에 구현했습니다.

### 2. 몬테카를로 시뮬레이션 조건 (Simulation Setup)
모형의 강건성\과 변수 선택 일치성을 검증하기 위해 다음과 같이 환경을 통제하여 100회 복제 스트레스 테스트를 수행했습니다.
* **오차항 시나리오**: 가우시안(Gaussian), 코시(Cauchy; 이상치 환경), 지수(Exponential; 비대칭 환경) 분포
* **차원 설정**: 표본 크기와 변수 개수가 일치하는 임계 영역($P = N = 100$) 및 고차원 환경($P = 200, N = 100$)

---

## 시뮬레이션 실증 결과 

| 평가 시나리오 ($N=100$) | 비교 모형 (Baseline) | 제안 모형 (Sparse QR) | 통계학적 거동 및 수치 해석 |
| :--- | :---: | :---: | :--- |
| **코시 오차항 (Cauchy)** | 이상치에 의한 왜곡 (MSE 급증) | **안정적인 MSE 유지** | 절대오차 기반 체크 손실 함수의 강건 추정 성능 입증 |
| **임계 영역 ($P=N=100$)** | 행렬 특이성으로 오차 증가 | **수렴성 및 추정 정확도 유지** | 근접 솔버의 정규화 제약을 통한 해의 안정성 확보 |
| **고차원 영역 ($P=200$)** | 과적합 발생 (FDR 85%~98%) | **FDR 37.91% 수준 통제** | 참 신호 복원(TPR=0.9967) 및 불필요한 변수 효과적 제거 |
